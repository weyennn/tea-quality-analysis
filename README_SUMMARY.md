## Ringkasan Proyek

- **Masalah:** Penilaian kualitas teh cenderung subjektif dan sulit distandarkan.
- **Pendekatan:** Analisis data sensor aroma untuk mengklasifikasikan dan mengevaluasi kualitas teh.
- **Insight Utama:** Data sensor mampu membedakan kualitas/jenis teh secara konsisten melalui fitur statistik.
- **Dampak:** Mendukung proses pengendalian kualitas dan standarisasi evaluasi mutu produk berbasis data.
